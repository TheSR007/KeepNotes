Using python to represent Lists

Python knows a number of 
compound
 data types, used to group together other values. The most versatile is the 
list
, which can be written as a list of comma-separated values (items) between square brackets. Lists might contain items of different types, but usually the items all have the same type.
  


lists also support indexing, slicing, concatenation but they are mutable




iteams can also be added at the last using append() and len() can also be used



[:] returns a shallow copy(new list)


Assignment to slices is also possible, and this can even change the size of the list or clear it entirely
  


nesting in list is also possible
