Strings in python are surrounded by either single quotation marks, or double quotation marks.
'hello' is the same as "hello"
print() is used to display them in stdout



Multiline Strings  
multiline string can be assigned to a variable by using three quotes(single/double)



Strings are Arrays 

Like many other popular programming languages, strings in Python are arrays of bytes representing unicode characters.
However, Python does not have a character data type, a single character is simply a string with a length of 1.
Square brackets can be used to access elements of the string.


we can loop through the characters in a string, with a for loop


To get the length of a string, use the len() function


To check if a certain phrase or character is present in a string, we can use the keyword in
returns True if present or not in returns False




Python has a set of built-in methods that you can use on strings.






To concatenate, or combine, two strings you can use the + operator.
To add a space between them, add a ' ' or " "



we cannot combine strings and numbers like this:(need to convert to str(variable))


we can combine strings and numbers by using the format() function using { } as placeholder

You can use index numbers { } or variable too and also by putting f at start of the string



Note: :.2f is used to display 2 floating point digit


To insert characters that are illegal in a string, use an escape character \_.
